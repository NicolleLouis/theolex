# Theolex front-end for Docker  
Use `theolex/front` as reference folder

## Build Image
- Run  in `theolex/front/`
<br>`docker build -t theolex-front:app .`

## Run Image in Dev mode
- Run  in `theolex/front/`
<br>`docker container run -it -p 3000:3000 theolex-front:app`
